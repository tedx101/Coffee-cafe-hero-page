# Myme — Coffee Café Landing Page

A calm, editorial-style landing page for **Myme**, a specialty coffee space in Laxmi Nagar, Nagpur. Built from the [Figma design](https://www.figma.com/design/Enydhi95ctXLrKGvaPSng1/Landing-Page-Coffee).

Plain HTML/CSS/JS — no build step, no dependencies.

## Run locally
Just open `index.html` in a browser, or serve it:

```bash
npx serve .
```

## Structure
```
index.html       Page markup
css/styles.css    All styling
js/main.js        Mobile nav toggle + scroll state
design.md         Full design spec (colors, type, layout, copy) — source of truth
```
